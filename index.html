本地部署

开发要求：

  • Node.js version 20.6.0+
  • pnpm version 9+

克隆代码仓库：

git clone https://github.com/DavidBlackCN/Docs-For-TUW.git

安装依赖：

pnpm install

首次启动开发服务前，先构建源代码：

pnpm build

启动开发服务：

pnpm docs:dev

使用

# start dev server
pnpm docs:dev
# build for production
pnpm docs:build
# preview production build in local
pnpm docs:preview
# update vuepress and theme
pnpm vp-update
