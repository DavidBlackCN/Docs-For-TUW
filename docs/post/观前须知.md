# **【观前须知】**
本贴同步发布于`Bilibili`，`MineBBS`，`KLPBBS`，以及`红石中继站`   
本帖为**MCARPG系列《亡灵战争》**的最新唯一官方发布贴  
原官方发布帖已随着**MCBBS**的~~坠机(2024年)~~而失效  
亡灵战争正传系列地图目前已经**停更**，制作组已经解散，后续更新维护的版本称之为**“纪念版”**  
**“纪念版”**包含两张地图：**《亡灵战争-弥赛亚》**和**《亡灵战争Ⅱ-血色黎明》**   
**《亡灵战争-弥赛亚》**为原亡灵战争Ⅰ的重制版，正式版发布于2022年   
**《亡灵战争Ⅱ-血色黎明》**为正传第二部，正式版发布于2017年  
新玩家的游玩顺序应为：**《亡灵战争-弥赛亚》->《亡灵战争Ⅱ-血色黎明》**
