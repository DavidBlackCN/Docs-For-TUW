# **【亡战系列中最为酣畅淋漓的战斗！】**
《亡灵战争·弥赛亚》是系列中第一个大量使用**NPC脚本**与**RC's Effect**的正传作品。  
强大的脚本程序和模型编辑能让我们能做出各种酷炫的**主动技能**，**武器装备**与**BOSS战**。  
在这全新的世界里，**死亡之影·凋灵**将会极为凶残！  
为了人类，流尽你的最后一滴血吧，被选中的弥赛亚！
:::

# **【自由搭配的主动技能】**
在本作中，玩家可以在战斗准备阶段根据接下来 敌人的特性，  
从七个主动技能中自由选择三个组成自己的技能组。  
而属于人类弥赛亚的终极技能**“降临·生命之辉”**，则将在游戏后期直接解锁！  
所有技能的冷却时间/持续时间都将在快捷栏中 以耐久度的方式实时显示。  
**所有技能均有音效，以及我们手工绘制的原创技能图标！**
